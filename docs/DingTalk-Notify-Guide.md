## ⚠️ 注意事项

### 能做什么：
- ✅ 发送文本消息给指定 userId
- ✅ 发送文件、图片
- ✅ 发送 Markdown 格式消息
- ✅ 群发给多个 userId

### 不能做什么：
- ❌ 通过姓名查找用户（需要 userId）
- ❌ 主动发起私聊（需要先有对话或知道 userId）
- ❌ 创建钉钉会议（需要额外权限）
- ❌ 发送群消息（需要群 ID）

### 使用限制：
- 钉钉有消息频率限制，避免短时间大量发送
- 工作时间内发送，避免打扰同事休息
- 重要消息建议确认对方收到

## 🛠️ 技术细节

### 配置文件位置

```
/home/admin/.openclaw/workspace/dingtalk-contacts.json
```

### 消息 API

通过 OpenClaw 的 `message` 工具调用钉钉 API：

```javascript
message {
  action: "send"
  channel: "dingtalk"
  target: "102"  // userId
  message: "消息内容"
}
```

### 自动记录 userId

当同事给迪克发消息时，会自动记录到：
```
/home/admin/.openclaw/workspace/dingtalk-contacts.json
```
