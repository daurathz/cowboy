# 报告技能 (Report Skill)

> 📊 自动收集数据、生成和发送报告

## 能力（规划中）

- 周报收集和汇总
- 数据报表自动生成
- 定期发送给相关人员
- 支持多种格式（Markdown、Excel、PDF）

## 工作流（目标）

### 周报汇总
```
1. 每周五 16:00 自动触发
2. 收集团队成员周报
3. 整理和汇总
4. 发送给负责人
5. 归档到知识库
```

### 数据报表
```
1. 按周期（日/周/月）触发
2. 从数据源拉取数据
3. 生成图表和分析
4. 发送给相关人员
```
