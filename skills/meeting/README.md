# 会议技能 (Meeting Skill)

> 📅 会议自动化 - 会前准备、会中记录、会后跟进

## 能力（规划中）

- 检测日历事件
- 会前自动提醒参会人
- 自动准备会议材料
- 生成会议纪要
- 跟踪会议待办

## 工作流（目标）

```
1. 检测日历 → 发现明天有会
2. 检查参会人 → 确认谁能参加
3. 发送邀请 → 钉钉通知 + 会议链接
4. 会前提醒 → 提前 1 小时提醒
5. 材料准备 → 自动整理相关文档
6. 会议纪要 → 会后自动生成
7. 待办跟踪 → 分配任务 + 跟进
```
