## 📋 标准与规范

### 身份认证与授权

- **[OAuth 2.0](https://oauth.net/2/)** - 授权框架标准
- **[OpenID Connect](https://openid.net/connect/)** - 身份层标准
- **[JWT RFC 7519](https://jwt.io/)** - JSON Web Token 规范
- **[RFC 9700](https://datatracker.ietf.org/doc/html/rfc9700)** - OAuth 2.0 安全最佳实践 (2025)

### API 安全

- **[OWASP API Security Top 10](https://owasp.org/www-project-api-security/)** - API 安全十大风险
- **[NIST SP 800-207](https://csrc.nist.gov/publications/detail/sp/800-207/final)** - 零信任架构
- **[CISA Zero Trust](https://www.cisa.gov/resources-tools/resources/zero-trust-maturity-model)** - 零信任成熟度模型

### 数据安全

- **[NIST SP 1800-16](https://www.nist.gov/publications/nist-special-publication-1800-16)** - 数据泄露防护
- **[ISO 27001](https://www.iso.org/standard/27001)** - 信息安全管理体系
- **[GDPR](https://gdpr.eu/)** - 通用数据保护条例
